# Provide the Names or Controls for the Prefabricated Models

Create a `character` vector of the names of the pre-fabricated (prefab)
models or a `list` of their controls.

## Usage

``` r
prefab_models_controls()

prefab_models()
```

## Value

`prefab_models`: `character` vector of model names.  
`prefab_models_controls`: `list` of model controls.

## See also

Helper functions for prefab models:
[`runjags models`](https://weecology.github.io/portalcasting/reference/runjags-models.md),
[`tsglm models`](https://weecology.github.io/portalcasting/reference/tsglm-models.md)

## Examples

``` r
  prefab_models_controls( )
  prefab_models( )
```
