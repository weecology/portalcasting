# portalcasting: Model and Forecast Portal Rodent Dynamics

Create a directory, populate it with data, build models, forecast rodent
populations, and visualize the results. The user can spin-up a local
lightweight sandbox or full-scale production environment from the same
code underlying the continuously deployed Portal Predictions Project
continuous integration system
<https://github.com/weecology/portal-forecasts> and website
<https://portal.naturecast.org>.
